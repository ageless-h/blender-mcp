# Addon Contract Template

## Contract Version
- Version: 0.1.1

## Entrypoints
- `addon_entrypoint`: Minimal addon readiness entrypoint
- `execute_capability`: Minimal capability execution entrypoint (PoC)

## Supported Blender Versions
- LTS: 3.6
- Latest: 4.x

## Compatibility Notes
- Contract validation currently requires `addon_entrypoint`.
- Additional entrypoints MAY be added in a forward-compatible way.
