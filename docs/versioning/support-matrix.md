# Support Matrix Policy

## Scope
Support includes all current Blender LTS releases and the latest stable release.

## Update Rules
- Update matrix at each MCP release
- Document compatibility results per version

## Data Files
- `docs/versioning/support-matrix.json`
- `docs/versioning/compatibility-results.json`

## Deprecation and Rollback
- Announce deprecation with timeline
- Provide migration guidance
- Keep rollback path for last supported LTS
