# Capability Catalog

## Grouping
- Scene management
- Object manipulation
- Geometry and modifiers
- Materials and shading
- Rendering and output
- Animation and simulation
- File import/export

## Catalog Fields
- name
- description
- required_scopes
- min_version / max_version
- limitations

## Minimum Capability Set
- scene.read (scopes: none)
- scene.write (scopes: scene:write)
