# Documentation and OpenSpec Organization

## Documentation Home
- All project documentation lives under `docs/` with domain-based subdirectories.

## OpenSpec Home
- All OpenSpec artifacts live under `openspec/`.
- Changes are stored in `openspec/changes/<change-name>/` and archived under `openspec/changes/archive/`.

## Rules
- Do not duplicate OpenSpec artifacts in `docs/`.
- Link from `docs/` to OpenSpec artifacts when needed.
